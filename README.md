BND-APP - extends Ionic App Base
=====================

## Setup

```bash
$ git clone https://github.com/itd-nd-gov/bnd-cp-app-ionic.git
$ cd bnd-cp-app-ionic
$ npm insatll
$ bower install

# test local
$ ionic serve

# test on emulator
$ ionic platform add ios
$ ionic build ios
$ ionic emulate ios
```
